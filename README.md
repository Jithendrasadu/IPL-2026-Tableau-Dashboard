# 🏏 IPL 2026 Tableau Dashboard

This project is a **sports analytics dashboard** built using Tableau to replicate a broadcast-style IPL match experience.

---

## 📊 Dashboard Overview

The dashboard includes:

* Match Summary (Batting & Bowling Insights)
* Live Points Table with rankings
* Orange Cap & Purple Cap leaders
* Next Match preview
* Match Insights (Chasing %, Toss Impact, Win Patterns)

---

## 🛠️ Tools & Skills Used

* Tableau
* Data Modeling
* Calculated Fields
* Table Calculations (INDEX, ranking logic)
* Dashboard Design (UI/UX)
* Data Storytelling

---

## 🎯 Key Highlights

* Designed a **clean and consistent UI** inspired by live sports broadcasts
* Built **dynamic calculations** for rankings and match insights
* Focused on **storytelling with data**, not just visualization
